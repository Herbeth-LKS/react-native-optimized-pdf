# Example App

This directory contains an example React Native app demonstrating the usage of `react-native-optimized-pdf`.

## Features Demonstrated

- Basic PDF viewing
- Custom navigation controls
- Cache management
- Error handling
- Progress tracking
- Page change events

## Usage

See [App.tsx](./App.tsx) for the complete example.

## Running the Example

1. Install dependencies in the root project
2. Import this component in your React Native app
3. Run your app on iOS

```tsx
import ExampleApp from './example/App';

export default ExampleApp;
```
